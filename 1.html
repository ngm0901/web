HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

Today, the Web is the most popular area in computing. HTML is the language that creates a web page, which is the minimum unit of the web. In that sense, HTML is the starting point for the web. The importance of HTML is growing, especially in recent years, as the web is being used in mobile or desktop application development.
